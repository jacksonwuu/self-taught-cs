# boot xv6

1、编译 xv6 内核（环境为 Ubuntu 20.04）

```shell
$ git clone git://github.com/mit-pdos/xv6-public.git
$ cd xv6-public
$ make
```

2、查看 kernel 中`_start`符号的位置

```shell
$ nm kernel | grep _start
```

3、启动模拟运行

```shell
$ make qemu-gdb
```

4、在 GDB 里查看程序运行情况

```shell
$ gdb
(gdb) break * 0x0010000c
(gdb) c
(gdb) info reg
```
